# How to accomplish special characters such as `é`:

1. <Ctrl> + <Shift> + `u`
2. utf-8 code, such as `00e9`
3. <CR>

* e / -- 00e9
* e : -- 00eb
* o / -- 00f3
* o : -- 00f6
* a / -- 00e1
* a : -- 00e4
* u / -- 00fa
* u : -- 00fc

# How to accomplish special characters the easy way in Gnome

1. Set compose key in keyboard settings
2. <Composer>, accent, letter

* ' for é
* ^ for ê
* ` for è
* " for ë
* =e for €
